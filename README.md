# Digital Clock

A modern, responsive, and aesthetically pleasing digital clock built with HTML, CSS, and Vanilla JavaScript. This project features a high-fidelity "7-segment" inspired design with a dark theme, highlighting the power of modern web technologies without the need for heavy frameworks.

## ✨ Features

- **Real-time Clock:** Precise time synchronization with your system clock.
- **12/24 Hour Toggle:** Switch between formats with a single click.
- **Dynamic Day Bar:** Highlights the current day of the week.
- **Full Date Display:** Shows the current month, day, and year.
- **Ultra Responsive:** Scales perfectly from mobile devices to large desktop monitors using fluid typography (`clamp()`).
- **Modern Dark Theme:** Deep black background with glowing white LED-style digits.
- **Zero Dependencies:** Pure Vanilla JS, CSS3, and HTML5.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, Safari, etc.).

### Installation

1. Clone or download this repository.
2. Navigate to the project folder.
3. Open `index.html` in your favorite web browser.

## 📂 Project Structure

- `index.html`: The structural skeleton of the clock.
- `style.css`: All the visual styling, including the dark theme and responsive logic.
- `script.js`: The "brain" of the clock, handling time updates and format toggles.
- `LEARN.md`: A detailed technical guide explaining the "how" and "why" behind the code—perfect for developers looking to learn.

## 🛠️ Technical Highlights

- **CSS Flexbox:** Used for perfect centering and alignment.
- **CSS `clamp()`:** Powers the fluid responsiveness.
- **JavaScript `setInterval`:** Drives the 1-second update loop.
- **Google Fonts:** Uses the 'Orbitron' font for a futuristic digital look.
